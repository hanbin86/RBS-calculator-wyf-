# RBS-calculator-wyf
Supplementary data and code for the undergraduate thesis. Feel free to download and bookmark!
以下文件注明：
name是菌种名单；
fetch_16s.py是抓取脚本
data.fasta是原始数据；
16SrRNA是多序列比对的数据
16SrRNA_fixed_v2.meg是修改格式等后的数据
16SrRNA_fixed_v2.mtsx是建树完以后的树数据（可用mega打开）
Newick Export.nwk是导出的树文件格式（用于ITOL可视化发育树）
